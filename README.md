## 📖 Описание

**NumRecog** - это система распознавания номеров объектов с эволюционной архитектурой, разработанная на Python с использованием PyQt6 для графического интерфейса и polars для эффективной обработки данных. Система реализует полный цикл обработки изображений: от загрузки и предварительной обработки до обнаружения объектов и распознавания цифр с помощью нейронной сети.

### Ключевые особенности

- **Эволюционная архитектура** - нейронная сеть адаптируется в процессе обучения с мутацией пороговых значений
- **Расширяемость** - заложена основа для перехода к резервуарному компьютеру (Reservoir Computing)
- **Многопоточность** - асинхронная обработка изображений без блокировки интерфейса
- **Совместимость** - поддержка оборудования 2012 года и старше (оптимизированные алгоритмы)

## ✨ Возможности

### Обработка изображений

| Функция | Описание | Методы |
|---------|----------|--------|
| **Загрузка** | Поддержка различных форматов | PNG, JPG, JPEG, BMP, TIFF |
| **Предобработка** | Улучшение качества изображения | Гамма-коррекция, логарифмическая, линейная, Retinex SSR/MSR |
| **Бинаризация** | Преобразование в черно-белое | Фиксированный порог, Отсу, адаптивная (средняя/гауссова) |
| **Разметка объектов** | Однопроходный алгоритм с ABC-маской | Выделение связанных компонентов |
| **Извлечение признаков** | Характеристики объектов | Площадь, периметр, центроид, компактность, эксцентриситет, моменты |

### Нейронная сеть

| Характеристика | Значение |
|----------------|----------|
| Архитектура | Многослойный персептрон (MLP) |
| Входной слой | 64 нейрона (8×8 пикселей) |
| Скрытые слои | 32 → 16 нейронов |
| Выходной слой | 10 нейронов (цифры 0-9) |
| Функция активации | Сигмоида / Пороговая / Линейная |
| Обучение | Эволюционное с обратным распространением |
| Мутация | Адаптация пороговых значений |

### Графический интерфейс

- Вкладки для отображения оригинального, бинарного и размеченного изображений
- Интерактивные настройки параметров обработки
- Асинхронная обработка с индикацией прогресса
- Текстовый вывод результатов в реальном времени
- Сохранение/загрузка обученных моделей

## 🚀 Установка

### Системные требования

- **Python**: 3.8 или выше
- **Операционная система**: Windows 7+, Linux, macOS 10.12+
- **Оборудование**: Поддержка оборудования 2012 года и старше
- **Оперативная память**: 512 MB (рекомендуется 1 GB)

### Установка из исходного кода

```bash
# Клонирование репозитория
git clone https://github.com/numrecog/numrecog.git
cd numrecog

# Создание виртуального окружения (рекомендуется)
python -m venv venv

# Активация виртуального окружения
# Windows:
venv\Scripts\activate
# Linux/macOS:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Установка пакета в режиме разработки
pip install -e .
